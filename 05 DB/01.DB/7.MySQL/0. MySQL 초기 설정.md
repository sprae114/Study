## workbench 사용법
[MySQL 설치 및 WorkBench 사용하기 (tistory.com)](https://dearmycode.tistory.com/15)
[MySQL 설치하기 | 얄코 (yalco.kr)](https://www.yalco.kr/@sql/3-1/)

```sql
--존재하는 DB 확인하기
show databases;

-- DB 생성하기
create database [생성할 DB];

-- 생성된 DB 사용하기
use [생성할 DB];
```

- 인텔리제이와 DB연결 설정
```yml
spring:  
  h2:  
    console:  
      enabled: true  
      
  datasource:  
    url: jdbc:mysql://localhost:3306/[생성할 DB]
    username: root  
    password: [생성할땐 만든 비번]
    initialization-mode: always
```

- 추가설정
```yml
spring:  
  h2:  
    console:  
      enabled: true  
  jpa:  
    show-sql: true  
    properties:  
      hibernate:  
        format_sql: false  
    generate-ddl: false  
    hibernate:  
      ddl-auto: create
```
